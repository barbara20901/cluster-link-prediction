This folder contains all the data files used:
  
  ### CITIBIKE HISTORICAL TRIPS ###
       -> cb_trips_18.csv.gz - citibike travel records in 2018 (preprocessed)
           *** citibike provides monthly data, this file is the full year concatenation ***
       -> stations_18.csv.gz - name, id, latitude and longitude for stations with trips in 2018 
       -> cb_trips_19.csv.gz - citibike travel records in 2019
           *** citibike provides monthly data, this file is the full year concatenation ***
       -> stations_19.csv.gz - name, id, latitude and longitude for stations with trips in 2019
  
  ### METEOROLOGY DATA - IEM ###
       -> meteo_nyc_18.csv.gz - weather data according to NYC ASOS station in 2018
       -> meteo_nyc_19.csv.gz - weather data according to NYC ASOS station in 2019
  
  ### STATION STATUS - OPENBUS ### 
       -> bikes_avail_18.csv.gz - stations status in 2018 
          *** openbus provides monthly data, this file is the full year concatenation ***
